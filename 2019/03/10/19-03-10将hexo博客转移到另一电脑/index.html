将hexo博客转移到另一电脑

  1. 拷贝或者将本机hexo的文件上传到GitHub保管（有一些博客是在博客分支上创建了一个hexo的分支，将代码保存，很值得参考）

  2. 在新的电脑上部署好hexo的运行环境

  3. 将hexo所有文件转移到新的电脑之上

  4. ssh-keygen -t rsa -C "2368143744@qq.com"命令生成 id_rsa密钥、id_rsa.pub公钥

    在GitHub中选择setting->SSH and GPG keys -> 右上角的New SSH key，配置好新电脑上的公钥

  5. git bash here进入hexo文件夹，开始进行hexo博客开发工作
